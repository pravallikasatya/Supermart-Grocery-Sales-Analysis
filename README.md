# 🛒 Supermart Grocery Sales Analysis

## 📌 Project Overview

This project performs end-to-end analysis of supermarket sales data, including data preprocessing, exploratory data analysis (EDA), visualization, and machine learning modeling to generate business insights.

---

## 📌 Business Problem

Retail businesses face challenges in understanding sales performance, forecasting demand, and optimizing inventory. This project helps solve these problems using data-driven analysis.

---

## 🎯 Objective

* Analyze sales performance across categories and regions
* Identify top-performing products
* Study monthly sales trends
* Build predictive models for forecasting

---

## 🛠 Tools & Technologies

* Python (Pandas, NumPy)
* Matplotlib & Seaborn
* Scikit-learn
* Jupyter Notebook

---

## 📂 Project Structure

* data/ → dataset
* notebooks/ → analysis code
* reports/ → detailed reports
* images/ → visualizations

---

## 📊 Exploratory Data Analysis

* Category-wise sales analysis
* Region-wise performance evaluation
* Monthly trend analysis
* Correlation analysis between sales and profit

---

## 🤖 Machine Learning Models

* Linear Regression (baseline model)
* Random Forest (better performance and accuracy)

---

## 📈 Key Insights

* Meat & Fish category generates the highest sales
* Sales show a consistent upward trend over time
* Strong positive correlation between sales and profit
* Significant regional variations in performance

---

## 💡 Business Impact

* Helps improve demand forecasting
* Supports better inventory management
* Enables data-driven pricing strategies
* Identifies high-performing products and regions

---

## 🚀 Conclusion

This project demonstrates a complete data science workflow from raw data processing to business insights and predictive modeling, showcasing real-world analytical skills.
