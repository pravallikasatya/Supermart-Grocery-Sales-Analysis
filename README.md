# Supermart-Grocery-Sales-Analysis
End-to-end data analysis and machine learning project on grocery sales dataset
📌 Business Problem
------------------------------------------------------------------------------------------------------
Retail businesses face challenges in understanding sales performance, forecasting demand, and optimizing inventory.

🎯 Objective
-------------------------------------------------------------------------------------------------------
Analyze sales performance
Identify top categories
Study trends
Build predictive models

🛠 Tools Used
---------------------------------------------------------------------------------------------------
Python (Pandas, NumPy)
Matplotlib & Seaborn
Scikit-learn

📂 Project Structure
=====================================================================================================
data/ → dataset
notebooks/ → analysis code
reports/ → reports
images/ → visualizations

📊 Key Insights
=====================================================================================================
Meat & Fish category highest sales
Sales increase over time
Regional variations exist

🤖 Machine Learning
=====================================================================================================
Linear Regression
Random Forest

🚀 Conclusion
=============================================================
End-to-end data analysis and machine learning project.
